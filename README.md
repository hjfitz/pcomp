# PlantUML Compiler
> Generate SVG and PNGs from `*.puml` files

```bash
$ yarn global add pcomp
```


## Usage:

Generate a SVG and PNG:

```
$ pcomp -h
Usage: index.js [options]
Options:
        --file=...              filename to compile
        -h, --help              show this dialog
        --png                   save as png
		--svg                   save as svg
```
